Just another plugin that creates shortcodes for embedding videos in posts.

== Description ==

There are a lot of plugins that allow users to embed videos using a shortcode.
Now there's one more out there.

Video Embeds currently supports YouTube and Vimeo.

Video Embeds allows for control over many of YouTube's embed paramaters and
all of Vimeo's embed parameters. Set default values for all available options
on the settings page.

Add FitVids.js to your site with the click of a button.

And a few other things too.

== Installation ==

1. Upload this plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Admin > Settings > Video Embeds to adjust settings
4. Start using the `[ve]` shortcode.
